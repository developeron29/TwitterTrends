# TwitterTrends
Twitter Trend analysis app

### Add Twitter credentials to let the app fetch real time tweets


File index.js, Line 8

```
var client = new Twitter({
    consumer_key: '',
    consumer_secret: '',
    access_token_key: '',
    access_token_secret: ''
});
```

## To run
```
node index.js
```

Note: Procfile included, Could be pushed to heroku directly
